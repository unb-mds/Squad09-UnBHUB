    Protótipo Inicial

    Usuário:
└── Nome: String
└── Email: String
└── Senha: String
└── Curso: String
└── Semetre: Integer
└── Materias:
|   └── Materia1:
|   |   └── Nome: String
|   |   └── Professor: String
|   |   └── DiasSemana: String
|   |   └── Horarios: Time
|   |   └── Local: String
|   |   └── Provas:
|   |   |   └──  P1:
|   |   |   |    └── Código: String
|   |   |   |    └── Nota: String
|   |   |   |    └── Data: String
|   |   |   |    └── Horário: Time
|   |   |   |    └── Sala: String
|   |   |   |    └── Conteudo: String
|   |   |   |    └── Status: String -> Completo, Por vir
|   |   └── Tarefas:
|   |   |   └── Tarefa1:
|   |   |   |   └── Nome: String
|   |   |   |   └── Descrição: String
|   |   |   |   └── DataEntrega: DateTime
|   |   |   |   └── Status: String -> Entregue, Atrasada, Em andamento
|   └── Livros:
|   |   └── Livro1:
|   |   |   └── Nome: String
|   |   |   └── DataEmprestimo: String
|   |   |   └── DataEntrega: String
|   |   |   └── Status: String -> Entregue, Atrasado, Lendo.
